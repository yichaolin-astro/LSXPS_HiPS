# LSXPS_HiPS
HiPS file for The Living Swift XRT Point Source Catalogue (LSXPS).

RGB false colored (0.3-1, 1-2, 2-10 keV).

Built using stacked images from LSXPS.

Last updated: Feb 22, 2026.

Access link: https://yichaolin-astro.github.io/LSXPS_HiPS/SwiftXRT_color

To access using ESASky or Aladin, paste the HiPS file link when prompted to add HiPS file from URL.

To download the HiPS file, see the release page.

# Known Issue:

Difficulty with seeing low flux count X-ray photons due to the scaling limit of physical pixel counts.
